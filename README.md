# UDPcomm

## Overview
Peer-to-peer UDP chat with IPv6 and Yggdrasil mesh support. Created for fun

## Usage
Build the project:
```bash
make
```
Run the executable:
```bash
./comm
```

## Features
- IPv6 via Yggdrasil mesh integration
- UDP holepunching
